{# Right-side live-calls drawer — four-section accordion frame. Sections (fixed order, only ONE expanded at a time): 1. Live Calls — currently in-flight, polled every 3s 2. Today's Calls — every ended call since midnight Mel, polled every 10s 3. Recently Searched — local jump-list of /customer/ visits in THIS browser (localStorage only — no server), refreshed on the ``recent-lookups-changed`` custom event card.html dispatches. 4. Pin Calls — agent-pinned calls, polled on pin-changed event Each section's body is HTMX-fetched from its own endpoint (except Recently Searched which renders client-side from localStorage). Drawer open/close is in the body's Alpine scope (set in base.html). #}
